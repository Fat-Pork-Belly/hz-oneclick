# Changelog

All notable changes to this project will be documented in this file.

## v3.0.0-alpha

### Added
- Horizon-Hybrid architecture direction: hybrid cloud delivery for Oracle Cloud & homelabs.
- Unified command/menu experience foundation (“Coke Menu”): a single entry surface for modules and presets.
- Deep Check (baseline-driven): preflight + post-install verification hooks for reliability and compliance.

### Changed
- Restructured modules layout to support segmented delivery (web/ops/net/media/backup) and future presets.
- CI/maintenance hardening: workflow lint stabilization and recovery process improvements.

### Fixed
- Sanitization and cleanup improvements from Phase 1 stabilization (removal of accidental patch artifacts, duplicate blocks, and syntax issues).
